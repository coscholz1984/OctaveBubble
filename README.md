# OctaveBubble

This is an implementation of the bubble shooter genre, well known by games such as bust-a-move (Puzzle Bobble) or Frozen Bubble, written in GNU Octave.

Run drawMain.m to start the game.

Controls are 
A - Left
D - Right
Space - Shoot
